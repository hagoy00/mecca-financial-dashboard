# Medhanialem Mekan Selam Ethiopian Church  
## Financial Dashboard (2021–2025)

A professional, board‑grade financial dashboard built for  
**Medhanialem Mekan Selam Ethiopian Church** to analyze, visualize, and report  
financial performance using an Excel‑only data pipeline.

This dashboard is designed for non‑technical finance teams and church board members  
who need clear, accurate, and accessible financial insights.

## 📊 Features

### **1. Board Summary (2021–2025)**
- Total Income  
- Total Expenses  
- Net Income  
- Year‑over‑year trends  
- Clean pivot table for board reporting  

### **2. YOY Summary**
- YOY % change for Income, Expenses, Net Income  
- Heatmap visualization  
- Trend charts  

### **3. Surplus & Deficit Analysis**
- Surplus/Deficit table  
- YOY change  
- Color‑coded indicators (green = positive, red = negative)  
- Multi‑year trend chart  

### **4. Forecasting (2026–2031)**
- Adjustable growth assumptions  
- Forecasted Income, Expenses, Net Income  
- Trend visualization  
- Interpretation summary  

### **5. Charts & Visualizations**
- Income trend  
- Expense trend  
- Net Income trend  
- Category breakdowns  
- Yearly comparison tables  

### **6. Subtotals by Category**
- Income or Expense subtotals  
- Year‑based filtering  
- Category‑level bar charts  

### **7. Board Report PDF Generator**
Generates a professional PDF including:
- Board summary  
- YOY summary  
- Forecast summary  
- **Top 5 Income (2021–2025)**  
- **Top 5 Expenses (2021–2025)**  
- Automatic page breaks and clean formatting  

### **8. Admin Data Upload**
- Upload updated Excel file  
- Must be named: `MECCA_Financial_Data.xlsx`  
- Dashboard updates instantly
